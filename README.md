# instagram_post_-to-_pdf

This is a python script to download multiple images of a single post in instagram and convert those images into a pdf file

requirements:

1. python
2. instaloader
3. this script

process:

1. you need to python installed on your machine.
2. install instaloader using a command ( pip install instaloader pillow )
3. then save this script in a file, and execute this script using command ( python script_file.py )
4. you will be asked to enter the link of the post.
5. after you enter the link, name the required pdf file name.
6. now images will be downloaded in a folder called "downloads" and each image will be saved with jpg format and named as pdf file name and a sequence number.
7. finally your pdf file will be generated where the script is saved.
